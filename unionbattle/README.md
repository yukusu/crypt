### ユニオンバトル周回用スクリプト
 
**※キーはすべて任意のキーに変更可能です。**   
**※座標は各環境に合わせて変更していただく必要があります。**  
**※画像は自前で用意して頂く必要がある可能性があります。**  
**※無限クリックになってしまった場合は無変換+rキーでスクリプトをリロード・無変換+pキーでポーズ・無変換+
